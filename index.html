Una herramienta util para divisar informacion fácil y ordenadamente. Realizar este proyecto demandó bastante empeño y dedicación ya que fue uno de los primeros que realizé. MENSAJES OCULTOS

Mensajes Ocultos es una aplicación web que recibe un texto y mediante el cifrado cesar te devuelve otro texto que dificilmente podrá ser entendido por alguien más. En este proyecto se utilizo javascript puro, el objetivo principal fue entender la lógica con la cual se hizo el cifrado, una buen forma de sumergirse en el mundo de la programación
